# Questions à poser à Gemini (Guide de Démonstration)

Pour tester la puissance de Jumeau sur une application web (Salesforce, Notion, GitHub, LinkedIn, etc.), voici ce que tu peux demander à l'IA :

## 1. Compréhension Visuelle & Identification
*   "Regarde cette page. Peux-tu pointer avec ton curseur les éléments qui te semblent être des données structurées exploitables par un workflow ?"
*   "Quels sont les trois champs les plus importants sur cet écran pour créer une fiche client ?"

## 2. Suggestions de Workflows (Skills)
*   "En te basant sur ce que tu vois, quel workflow n8n pourrais-tu construire pour m'aider à gagner du temps ici ?"
*   "Imaginons que j'aie un serveur MCP connecté à mon Slack. Comment pourrais-tu envoyer les informations de cette page vers le channel #veille ?"

## 3. Interaction & Clarification
*   "Si je te demande de 'nettoyer' ces données, sur quel élément aurais-tu besoin de me poser une question de clarification ? Montre-le moi avec le curseur."
*   "Crée un Blueprint de workflow qui prend ce contact et l'envoie vers HubSpot. S'il manque une information visuelle, demande-la moi."

## 4. Export & Transformation
*   "Génère un fichier `.md` de Skill qui explique comment automatiser la lecture de ce dashboard pour un nouvel utilisateur."
*   "Peux-tu transformer cette vue de liste en une structure JSON prête à être injectée dans mon webhook n8n ?"

---

> **Tip** : N'hésite pas à être vague au début ("Fais un truc avec ça") pour voir comment Jumeau utilise la vision pour deviner ton intention !
